# Approval Checklists
**System:** MindSpark MusicWorks™

This folder holds the final signed approval documents for every completed project. It is the permanent record that each release was reviewed, approved, and launched intentionally.

## What Goes Here

- Completed Founder Approval Dashboards (from Agent 13)
- Signed Theology Approval Certificates (from Agent 03)
- Signed Audio QC Certificates (from Agent 04)
- Signed Founder's Declaration
- Any second-approver co-signatures

## Naming Convention

`[project_name]_founder_approval_[date].md`
`[project_name]_theology_certificate_signed.pdf`
`[project_name]_audio_qc_certificate_signed.pdf`

## Rule: Nothing Ships Without This

The approval folder for a project must contain a signed Founder's Declaration before that project is considered launch-ready. This is the most important folder in the system.

If you are looking for a project's approval status, check here first.
