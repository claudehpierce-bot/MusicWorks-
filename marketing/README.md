# Marketing
**System:** MindSpark MusicWorks™

This folder holds all approved marketing assets and campaign files for each release.

## What Goes Here

- Approved social media captions (from Agent 06)
- Short-form video content calendars and scripts (from Agent 07)
- YouTube strategy documents and upload schedules (from Agent 08)
- Playlist outreach trackers (from Agent 09)
- Press releases (from Agent 06)
- Email campaign sequences (from Agent 10)

## Rule

Only approved content lives in this folder. Drafts and un-reviewed material stay in the project's `/fire_and_flow_validation/[project]/agent_outputs/` folder until the founder approves them and they are moved here.
