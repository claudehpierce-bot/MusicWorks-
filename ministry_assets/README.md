# Ministry Assets
**System:** MindSpark MusicWorks™

This folder holds all ministry and devotional resources created for each release — the tools that turn albums into ministry partners.

## What Goes Here

- 7-Day Devotional Guides (from Agent 11)
- Small Group Discussion Guides (from Agent 11)
- Sermon Illustration Notes (from Agent 11)
- Prayer Cards (from Agent 11)
- Choir Arrangement notes (when applicable)
- Church Licensing information
- CCLI registration records

## Subfolder Structure Per Project

```
/ministry_assets/
  /becoming_vol_1/
    7_day_devotional_becoming.md
    small_group_guide.md
    sermon_illustrations.md
    prayer_cards.md
    licensing_info.md
```

## Distribution of Ministry Assets

Ministry assets are intended to be given freely to churches, small groups, and individual believers. They may also be offered as downloads in exchange for an email list sign-up. Before distributing, confirm:

- [ ] Theological review signed
- [ ] Founder approved
- [ ] Correct scripture version cited throughout
- [ ] Contact information for the ministry included

## Note on Transparency

Ministry assets must be honest about being AI-assisted in their creation. They should carry a note such as: *"Prepared with AI assistance by MindSpark MusicWorks™. Reviewed and approved by [Name], [Title]."*
