# Quality Control
**System:** MindSpark MusicWorks™

This folder holds QC reports, signed certificates, and re-check records for every project.

## What Goes Here

- Completed Audio QC checklists (from Agent 04)
- Signed Audio QC Certificates (from the audio engineer)
- Re-check records when files fail and are corrected
- Artwork QC reports
- Any third-party QC reports (mastering engineer notes, etc.)

## Naming Convention

`[project_name]_audio_qc_[date].md`
`[project_name]_artwork_qc_[date].md`
`[project_name]_qc_certificate_signed.pdf`

## Why This Folder Matters

Distribution platforms will reject or silently degrade audio that does not meet technical standards. A signed QC certificate is proof that human ears checked the work before it shipped. This folder is the audit trail.
