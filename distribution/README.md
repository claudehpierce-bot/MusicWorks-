# Distribution
**System:** MindSpark MusicWorks™

This folder holds all metadata packages, distribution checklists, and submission records for each release.

## What Goes Here

- Master metadata spreadsheets (from Agent 05)
- Distribution platform checklists (from Agent 05)
- PRO and royalty registration records (from Agent 05)
- Distribution submission confirmations
- ISRC and UPC/EAN records
- Split sheet records (copies — originals should be held in secure storage)

## Subfolder Structure Per Project

```
/distribution/
  /becoming_vol_1/
    metadata_master.md
    distribution_checklist.md
    pro_registration_record.md
    isrc_log.md
    submission_confirmations/
```

## Critical Rule

This folder is the most legally sensitive folder in the system. Every document here relates to credits, ownership, and royalty payments. Accuracy is non-negotiable. Never edit a metadata document after submission without documenting the change and reason.
